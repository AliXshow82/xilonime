# 🇺🇸 English

Xilonimeh is a desktop application designed for managing personal anime lists in a fast, organized, and modern environment.

The app helps users track:
- Watching anime
- Completed anime
- Paused anime
- Planned anime

while keeping everything stored locally on the user's system.

## ✨ Features

- Local data storage
- Lightweight and modern UI
- Custom frameless window
- Built-in Persian Vazir font
- Anime status categories
- Search, filtering, and sorting system
- Anime statistics and watched episode tracking
- Designed for Windows desktop
