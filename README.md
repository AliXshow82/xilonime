# ژایلونیمه

ژایلونیمه یک برنامه دسکتاپ برای ساختن و نگهداری لیست شخصی انیمه‌هاست.

## ویژگی‌ها

- ذخیره‌سازی محلی روی سیستم کاربر
-- پنجره سفارشی بدون قاب پیش‌فرض ویندوز
- فونت فارسی Vazir به صورت محلی
-- دسته‌بندی وضعیت انیمه‌ها مثل در حال تماشا، تمام‌شده، برنامه آینده، متوقف‌شده و رهاشده
- جستجو، فیلتر و مرتب‌سازی لیست انیمه‌ها
- نمایش آمار کلی لیست و قسمت‌های دیده‌شده

## اجرا برای توسعه

```bash
npm install
npm start
```

## ساخت فایل نصبی

```bash
npm run dist
```

خروجی نصب‌کننده:

```text
dist/Xilonimeh-Setup-0.2.0-alpha.0.exe
```
